# HTML Card Layout
This project is a simple card layout created using HTML and inline CSS.
Multiple cards are displayed in a single row using the Flexbox layout. Each card contains a title, description, and a button.
The main objective of this project is to practice basic HTML structure and understand how to arrange elements horizontally using Flexbox while maintaining spacing between cards.
This project was developed as part of my HTML learning journey at **10000 Coders** while strengthening my front-end development fundamentals.

## Technologies Used
* HTML5
* CSS (Inline Styling)
* Flexbox
